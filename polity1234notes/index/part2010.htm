11.1. Polling Stations

Each constituency is divided into a number of polling areas. For each polling area, one or more polling stations are provided according to the number of voters.

Outside each polling station there will be displayed prominently a notice specifying the polling area, the electors of which are entitled to vote at the polling station and a list of the contesting candidates.
